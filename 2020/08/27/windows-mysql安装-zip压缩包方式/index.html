windows mysql安装-zip压缩包方式

发表于 2020-08-27

安装流程

  • 官网下载zip安装包，并解压

  • 根目录创建配置文件

  • 管理员权限打开cmd，并进入mysql的bin目录

  • 初始化mysql，mysqld –initialize-insecure （建议使用，不设置root密码），执行完会自动创建data目录

  • 安装mysql服务

  • net start mysql开启Mysql服务

  • 登录mysql

  • 修改root密码

  • 退出mysql

  • 重新登录mysql

环境变量配置

  • 新增MYSQL_HOME
  • 增加到Path
