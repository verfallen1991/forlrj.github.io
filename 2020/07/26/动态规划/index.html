基本题型 leetcode 322 零钱兑换

思路

  • 自顶而下看，存在重叠子问题
  • 子问题存在最优解

解法

  • 状态转移方程
    d[n] = min(d[n-x], d[n-y], d[n-j]) + 1
  • 自底向上求解

基本题型 leetcode 368 最大整除子集

  • 状态转移方程：dp[i] = max(dp[j] + 1) 对于所有的[0,j)，有nums[i] % nums[j] == 0
    注：通过此方程，dp[i]的值是不依赖与后面元素的，所以可以使用dp
  • 遍历每个位置都在pre数组中记录其最大整除子集剔除自己后的最大元素位置
